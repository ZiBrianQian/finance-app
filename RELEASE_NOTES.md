# Release v1.4.7

## Исправления

### Денежные значения в интерфейсе
- Исправлено отображение больших сумм на сводке, счетах, бюджетах, целях, долгах, транзакциях, отчетах, подписках и календаре.
- Суммы больше не переносятся посередине числа и не ломают карточки.
- Для компактных мест добавлен аккуратный truncate, а для KPI-карточек - динамический размер суммы без увеличения самих карточек.
- Исправлено отображение блока "Топ расходов" и компактного preview счетов в sidebar/dashboard.
- У графиков увеличена область оси Y, чтобы крупные числа не обрезались слева.

### Проверка обновлений
- Исправлена ошибка `releases.filter is not a function`, когда GitHub API возвращает не список релизов, а объект ошибки.
- Добавлена понятная обработка GitHub API rate limit.
- Автопроверка обновлений при старте теперь использует кеш, чтобы не расходовать лимит GitHub API при каждом запуске.
- Если GitHub вернул rate limit, приложение запоминает время reset и не повторяет запрос до него.

## Проверка

- Production build: `npm run build`
- Portable build: `npm run electron:build`
- Проверено вручную на тестовых данных с большими суммами.
